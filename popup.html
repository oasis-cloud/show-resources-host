说明

安装本插件后，可在激活的 Tab 选项卡中显示页面源码所包含的 script、link 标签 HostName 的 IP 地址。

开启默认配置后，在激活的 Tab 选项卡中将显示页面源码所包含的 script、link 标签 HostName 的 IP 地址，以及显示单独设置的HostName。

关闭默认配置后，在激活的 Tab 选项卡中将不再显示页面源码所包含的 script、link 标签 HostName 的 IP 地址。仅仅显示单独设置的HostName。
